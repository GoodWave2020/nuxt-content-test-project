利用可能なコンポーネント

Alert

情報、警告、エラー、成功メッセージの表示

::alert{type="info"}

Card

カード形式のコンテナ、カラーバリエーション付き

::card{title="タイトル" icon="📝" color="blue"}

FeatureCard

機能を紹介するためのカード

::feature-card{title="機能" icon="🚀" color="purple"}

コンポーネントの定義方法

1️⃣ Vueコンポーネントファイルの配置

コンポーネントは components/content/ ディレクトリに配置します。

components/
└── content/
    ├── Alert.vue
    ├── Card.vue
    └── FeatureCard.vue

2️⃣ コンポーネントの定義例

例: components/content/Card.vue

<script setup lang="ts">
defineProps<{
  title?: string
  icon?: string
  color?: 'blue' | 'purple' | 'green' | 'red'
}>()
</script>

<template>
  <div :class="['card', color]">
    <div class="icon"></div>
    <h3></h3>
    <slot />
  </div>
</template>

3️⃣ MDC構文での呼び出し

ファイル名がMDC構文にマッピングされます（PascalCase → kebab-case）

::card{title="タイトル" icon="🔵" color="blue"}
カードのコンテンツ（slotに渡される）
::

4️⃣ 名前のマッピング規則

  • Card.vue → ::card
  • FeatureCard.vue → ::feature-card
  • Alert.vue → ::alert

5️⃣ Props と Slots

  • Props: {prop="value"} の形式で渡す
  • Slots: ::component-name と :: の間のコンテンツ
  • 自己閉じタグ: スロットなしの場合は ::component-name{props}
